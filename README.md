# Samsung Internet A-Frame Demos

## Set up

The demos are collected together as a Jekyll site. To get it up and running...

### Linux

```bash
sudo apt-get install bundler zlib1g-dev libxml2-dev nodejs
bundle install
bundle exec jekyll serve
```
